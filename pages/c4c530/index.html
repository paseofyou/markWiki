  • 作用: 美化页面

# 添加css样式

  • 三种引入方式

    引入方式 释义 优点 弊端 代码
    内联 在标签的style属性中添加样式代码 不能复用 <h1 style="color: red">内联样式</h1>
    内部 :在head标签里面添加style标签,标签体内写样式代码,通过选择器先找到元素对象, 然后再添加样式, 可以复用 只能在当前页面复用,不能多页面复用 <head>
    <style>
    h2{color: blue}
    </style>
    </head>
    <body>
    <h2>内部样式</h2>
    </body>
    外部 在单独的css样式文件中写样式代码, 在html页面中引入此文件 多页面复用，推荐 <head>
    <link rel="stylesheet" href="my.css">
    </head>
    <body>
    <h3>外部样式</h3>
    </body>

选择器 作用 代码格式 示例
标签名选择器 匹配页面中所有同名标签 标签名{样式代码} <head>

PX是Pixel的缩写，也就是说像素是指基本原色素及其灰度的基本编码，由 Picture (图像) 和 Element（元素）这两个单词的字母所组成的

# CSS颜色赋值

  • 三原色: 红绿蓝 RGB Red Green Blue , 每个颜色的取值范围0-255
  • 五种颜色赋值方式:
    • 颜色单词赋值: red/yellow/pink/purple.....
    • 6位16进制赋值: #ff0000
    • 3位16进制赋值: #f00 每一位重复一次 等效ff0000
    • 3位10进制赋值: rgb(255,0,0)
    • 4位10进制赋值: rgba(255,0,0,0-1) a=alpha 设置透明度 值越小越透明
    	<style>
     	h1 {
          color: red;
          color: #00ff00;
          color: #00f;
          color: rgb(255,0,0);
          color: rgba(255,0,0,0.5);
      	}
  	</style>

# CSS背景赋值

  • width,height设置元素的宽高

    • 默认宽高:宽度是上级元素宽度的100%,高度是内容高度
    • px代表像素 除了0其它数值都必须写px

  • background-image:url("资源路径"); 设置背景图片

  • background-size: 宽度 高度; 设置背景图片尺寸

  • background-repeat: no-repeat; 禁止重复

	<style>
        div{
            width: 200px;
            height: 200px;
            background-color: red;
            background-image: url("../b.jpg");
            background-size: 100px 100px;
            background-repeat: no-repeat;
        }
    </style>
  • background-position: 横向偏移值 纵向偏移值; 设置背景图片的位置,可以将偏移值换成偏移的百分比
	<style>
        background-position: 0px 50px;
		background-position: 0% 100%;
    </style>

# 文本和字体相关样式

  • text-align:left/right/center; 设置水平对齐方式

  • text-decoration: 样式;

    • overline上划线
    • underline下划线
    • line-through删除线
    • none去掉文本修饰; 设置文本修饰

  • text-shadow:颜色 x偏移值 y偏移值 浓度; 设置文本阴影

        <style>
        div{
            text-shadow: red -20px -10px 1px;
        }
    </style>

  • line-height:20px; 设置行高,默认高度包裹文本,。设置元素的高度可以实现垂直居中

  • font-weight:bold; 设置加粗;normal去掉加粗

  • font-style:italic;设置斜体

  • font-size:20px; 设置字体大小

    • 修改时，会自动将margin-top和margin-bottom改为font-size的值

  • font-family: xxxx,xxx,xxx; 设置字体。font-family: cursive;

  • font: 30px xxx,xxx,xxx; 设置字体大小+字体。font: 20px cursive;

  • border-radius: 1px设置圆角

# 设置元素的显示方式 display

  • block: 块级元素, 特点: 独占一行并可以修改宽高, 包括:h1-h6,p,div
  • inline: 行内元素, 特点: 共占一行,不能修改宽高（需改成其他显示方式，块级或行内块 ）, 包括:span,b,i,u,s,超链接a
  • inline-block:行内块元素,特点: 共占一行并可以修改宽高,包括:input,img
  • none: 隐藏元素

# 盒子模型

  • 盒子模型= content内容+margin外边距+border边框+padding内边距

  • 盒子模型用来控制元素的显示效果:

    • content内容: 控制内容的显示宽高
    • margin外边距: 控制元素的显示位置
    • border边框: 控制边框效果
    • padding内边距: 控制元素内容的位置

# 盒子模型:Content内容

  • 用来控制元素的显示宽高
  • 相关样式:
    • width 设置宽度
    • height 设置高度
  • 两种赋值方式: 1. 像素 2. 上级元素的百分比
  • 行内元素不能修改宽高,如果必须修改则将元素改成块级或行内块

# 盒子模型:Margin外边距

  • 作用:用来控制元素显示的位置
  • 元素距上级元素或相邻兄弟元素的距离称为外边距
  • 赋值方式:
    • margin-left/right/top/bottom:10px; 单独某一个方向添加外边距
    • margin:10px; 四个方向添加外边距
    • margin:10px 20px; 上下和左右赋值 左右auto时为居中
    • margin:10px 20px 30px 40px; 上右下左顺时针赋值
      • 上下相邻，彼此添加外边距，取两者最大值。
      • 左右相邻，彼此添加外边距 两者相加
  • 行内元素上下外边距无效，左右外边距两者相加
点击查看 
<style>
	#s1{
        border: 1px solid red;
        margin-right: 50px;
    }
    #s2{
        border: 1px solid blue;
    }
</style>
  • 粘连问题:
    • 当元素的上边缘和上级元素的上边缘重叠时,给元素添加上外边距会出现粘连问题
    • 给上级元素添加overflow:hidden解决子元素问题

# 盒子模型:border边框

  • 作用:控制边框效果
  • 赋值方式:
    • border:粗细 样式 颜色; 四个方向添加边框
    • border-left/right/top/bottom:粗细 样式 颜色; 单独某一个方向添加边框
  • border-radius:10px; 设置圆角 值越大越圆，超过宽高一半时为正圆

边框样式查询：

点击查看
值 描述
none 定义无边框。
hidden 与 "none" 相同。不过应用于表时除外，对于表，hidden 用于解决边框冲突。
dotted 定义点状边框。在大多数浏览器中呈现为实线。
dashed 定义虚线。在大多数浏览器中呈现为实线。
solid 定义实线。
double 定义双线。双线的宽度等于 border-width 的值。
groove 定义 3D 凹槽边框。其效果取决于 border-color 的值。
ridge 定义 3D 垄状边框。其效果取决于 border-color 的值。
inset 定义 3D inset 边框。其效果取决于 border-color 的值。
outset 定义 3D outset 边框。其效果取决于 border-color 的值。
inherit 规定应该从父元素继承边框样式。

# 盒子模型:Padding内边距(填充，内边距)

  • 作用: 控制元素内容的位置
  • 赋值方式: 和外边距类似
    • padding-left/right/top/bottom:10px; 单独某个方向赋值
    • padding:10px; 四个方向赋值
    • padding:10px 20px; 上下和左右赋值
    • padding:10px 20px 30px 40px; 上右下左顺时针赋值
  • 默认情况下
    • 给元素添加内边距会影响元素的宽高
    • 给元素添加box-sizing:border-box;后则不会影响宽高
