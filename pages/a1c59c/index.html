# 表设计面试题

2021年过年时小明在这些天都收到了许多亲戚\朋友还有同事的红包,也发出了一些红包,有的是微信,有的是支付宝也有现金,请参考下面的题目帮小明设计表格保存红包的信息

  1. 设计表 (至少包含一张流水表)

  • 列出需要保存的数据有哪些?

    • 人物关系, 红包金额,红包类型,时间,性别,名字
    • 由于名字，性别，人物关系数据固定，容易冗余，故划分为两张表（流水表trade，人物表person）

  • 创建流水表trade: 金额,时间, 红包类型,人物id

    create database Test charset=utf8;

use Test ;

CREATE TABLE trade
(
    id       INT PRIMARY KEY AUTO_INCREMENT,
    money    INT,
    personId INT,
    type     VARCHAR(30),
    time     DATE
) CHARSET = utf8;

  • 创建人物表:id,姓名,性别,关系

    CREATE TABLE person
(
    id     INT PRIMARY KEY AUTO_INCREMENT,
    name   VARCHAR(30),
    gender VARCHAR(2),
    rel    VARCHAR(10)
) CHARSET = utf8;

  • 往人物表中插入数据

    insert into person values(null,'刘德华','男','亲戚'),(null,'杨幂','女','亲戚'),(null,'马云','男','同事'),(null,'特朗普','男','朋友'),(null,'貂蝉','女','朋友');

  1. 统计2021年3月15号到现在的所有红包收益

    select sum(money) from trade where time>'2021-3-15';

  2. 查询2021年2月15号到现在 金额大于100 所有女性亲戚的名字和金额

    select name,money
from trade t join person p 
on t.p_id=p.id
where time>'2021-2-15' and abs(money)>100 and gender='女' and rel='亲戚';

SELECT p.name, t.money
FROM trade t,
     person p
WHERE t.personId = p.id
  AND ABS(t.money) > 100
  AND p.gender = '女'
  AND rel='亲戚';

  3. 查询三个平台(微信,支付宝,现金)分别收入的红包金额

    SELECT SUM(money), type
FROM trade
WHERE money > 0
GROUP BY type;
