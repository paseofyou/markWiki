# HTML

  • 作用: 学习如何搭建页面结构和准备页面的内容
  • HyperTextMarkupLang: 超文本标记语言
  • 超文本: 指不仅仅是纯文本,还包括字体效果和多媒体(音频,视频和图片)相关
  • 学习HTML : 有哪些标签,标签的使用方法

# HTML页面结构

  1. <!DOCTYPE html> 文档声明: 告诉浏览器使用HTML哪个版本的标准解析页面,此写法代表使用html5的标准解析页面
  2. <html lang="en">页面的根标签,所有标签都写在此标签内部
  3. <head></head> 头标签, 里面是给浏览器看的内容
    1. <meta charset="UTF-8"> 设置页面字符集
    2. <title>Title</title> 设置页面的标题
  4. <body> 体标签, 里面是给用户看的内容

针对IDEA编译器： <h1:以<开头，能够提示字母写法（仅此而已） h1: 不以<开头，能够自动生成标签属性 ctrl+D:复制整行到下一行

# 图片标签 img

idea里，输入<img后直接敲回车，可直接填充src属性

  • src：图片资源路径
    • 相对路径: 访问站内资源使用
      • 图片和页面在同级目录: 直接写图片名
      • 图片在页面的上级目录: ../../图片名
      • 图片在页面的下级目录: 文件夹/文件夹/图片名
    • 绝对路径: 一般访问站外资源使用, 又称为图片盗链, 有找不到图片的风险（直接对外网图片复制图片http路径）
  • alt: 图片不能正常显示时显示的文本
  • title:鼠标在图片上悬停时显示的文本
  • width/height:设置图片的宽高, 有两种方式:
    1. 像素 100
    2. 百分比 50%（只设置宽度width时高度会自动等比例缩放，只设置高度height就不会）

# 超链接标签 a

idea里，输入a敲tab键，可以直接补全href属性等

  • 图片超链接
<a href="http://www.tmooc.cn">
  <img src="a.jpg">
</a>

  • 页面内部跳转

    • 在目的地元素里面添加id,然后在超链接的href属性里面#id 这样就可以跳转到目的地元素的位置
    <p id="top">跳转到这里</p>
<br>
<a href="#top">点击跳转</a>

# 分区标签

  • 作用:

    • 可以将多个有相关性的标签添加到一个分区标签里面
    • 对多个标签进行统一管理
    • 可以将分区标签理解为一个容器

  • 分区标签

    • div： 块级分区标签，独占一行

    • span： 行内分区标签，共占一行

  • html5的标准中新增了一些分区标签,提高了代码的可读性(了解)

    • header 头标签
    • footer 脚标签
    • main 主体区域标签
    • nav 导航标签
